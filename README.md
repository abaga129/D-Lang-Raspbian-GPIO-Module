## LIBRARY INFORMATION

Raspbian library for accessing GPIO pins on a Raspberry PI.

Relies on the Linux GPIO SYSFS interface: 

https://www.kernel.org/doc/Documentation/gpio/sysfs.txt

http://elinux.org/RPi_GPIO_Code_Samples#sysfs
